Operators in Python¶

Python provies a wide range of possibilities for what we can do with our data. Most of the time, it is pretty intuitive on how to use them.

To manipulate data we are using something called an oeprator. An operator is a symbol that will carry out a specific function, such as a arithmetic or logical computation. They can be used on variables and values within Python.

An easy example of this is a simple mathematical computation:

In [3]:
z = 1 + x
print(z)

6
In [4]:
y = z * 4
print(y)

24

Where the plus, multiplication signs are our arithmetic opertors! Our equal sign in this case is what we call an assignment operator - it assigns the value of the equation to the right of the equal sign, into the variable on the left.

Within an IDE, these operators will show up as a different color than other commands, so they should be easy to spot.

When thinking of how this can be applied to data cleaning, it can be helpful if you want to center your data, create a new variable out of existing ones, find the mean of a variable, etc.

(I can come back to this example later when I am talking about libraries)

There are also other types of operators which allow you to complete different functions. For a full list of operators and what they do, this website has a comprehensive list of the ones you may encounter within Python.

For loops within Python¶

For loops in Python work a lot like for loops within bash. The same concept applies, though there are slight differences in the syntax of it all.

A quick refresher about for loops and their logic. We want to use for loops when we want our program to loop through a list of things. It can be used for a lot of different reasons; too many to even list here! You can use them in creative ways to help you automate tasks that you would otherwise do by hand.

Let's look at an example in Python.

In [10]:
majors = ["computer science", "psychology", "neuroscience"]
for x in majors:
    print(x)

computer science
psychology
neuroscience

In this case, there are three text variables contained within our list variable, "majors". Our for loop says that for every variable contained within the list, it should print each one.

You do not necessarily need to use 'x' to denote your variables in this case. You can use almost anything if it helps you make sense of your code later on:

In [11]:
majors = ["computer science", "psychology", "neuroscience"]
for words in majors:
    print(words)

computer science
psychology
neuroscience

This should be reminicent of what you leanred during the bash portion of the workshop. The only real difference between Python and bash is that you need to use a colon when setting up the for loop, and you do not need to close it by using "done".

Functions in Python¶

Sometimes when you are coding, you have something that you may want done many times. Rather than code this multiple times in multipe different places, you can create a function which will call a line of code. You can make a function as simple or complex as you would like, and the ability to pass different data into these functions can help shorten your code.

In [14]:
def my_function():
    print("I pass butter")
    
my_function()

def my_instructions():
    print("Raise your hand if you need help!")
    
my_instructions()

I pass butter
Raise your hand if you need help!

As you can see, any time that you type my_function() after you define it, it will perform that line of code. This can be helpful if you need to do something to your data multiple times.

You can also make it so you can pass variables into these functions to do the same computation on different variables.

iq_score = [100, 89, 72, 108, 120, 78, 86, 91, 95, 82]

    avg_variable = sum_variable/num_variable
    print("The average is:")
    print(avg_variable)

Data Manipulation in Python¶

The best way to show you all how to use Python to clean your dataset, is to use a real dataset!

We are going to be using some real data that Luz has graciously allowed us to use.

The first thing we will need to do is read our data into a format that Python can work with.

In order to do this, we need to use something called a software library. A library holds different commands and tools one can use, that are generally developed by people in an open source manner.

The library we are going to use to load in our data is called Pandas. This library is specifically aimed at data manipulation and analysis, so it’s a library you will encounter often.

To import Pandas, type:

In [ ]:
import pandas

This command is the most basic way of calling a library within Python. It calls the entire library and nothing else. There is one other way to do this though:

In [ ]:
import pandas as pd

The reason for the “as pd” portion of this command is to name our library something that is easy to be called on. In theory, you can call Pandas as whatever you would like. However, many people denote Pandas as pd, and if you are sharing your scripts with others, it would be beneficial to keep this consistent in your scripts.

To load in your dataset, type this command (making sure to edit the path file to reflect your computer and where you stored the data):

In [ ]:
#for Windows users
df_run1 = pd.read_csv(‘/Users/tuj96493/Downloads/DE249/DE249_dbdm_run_1.csv’)

#for Mac users

With this command, we are loading our first CSV file into our variable, called ‘df_run1’. It is important to name your dataset something meaningful so that when you are coding a script, it is obvious to you which variable you are handling.

This df_run1 variable is now our DataFrame. This is exactly what you would imagine it to be - a 2D representation of our tabular data. It is important to understand the way that Python stores this data, as it will impact the way we call on it.

Whatever your column and names are, Python will automatically bring those over from your data for you. So, it is important that the first row of your data are your variable names, so that you can easily find and manipulate the variables you want.

We can check our what is inside of our variable, and whether it was loaded correctly:

In [ ]:
print(df_run1)

This will give you information about header names, number of rows and columns, and other relevant information. You can also print certain rows or columns:

In [ ]:
print(df_run1[‘assignedITI’]

print(df_run1.loc[1])

The first command here is printing the column ‘assignedITI’. This includes all of the data points across trials for ‘asssignedITI’. The second command prints out a specific row that we may want to look at within the data.

This second command takes advantage of something in Python called indexing. Indexing is a way to refer to individual items within a DataFrame or list. Python is a language that uses objects that are “object-indexed”. This means that the first variable is coded at position 0 (as opposed to position 1, as you would think of in normal logic). This is a common thing to see in coding languages. So, if you want to refer to the first item in a row, you would have to use the command:

In [ ]:
print(df_run1.loc[0])

This item is the very first row of our dataframe. This does not include our column names.
